amayamaō vi hu dāhaṁ jaṇēi, sañjīvaṇō vi mōhēi

kuppanti vallahatthē apahuppantassa niyayaaṅgassa
annapahuppantassa ya puṇō asūyanti mihuṇāiṁ // SubhāPajjSaṁ.8

aṭṭhāṇaṭṭhanilayaṇaṁ visamiyasayaṇaṁ khalantamiuvayaṇaṁ

vallahajaṇassa dukkhē nisuē virahammi jāyaē sukkhaṁ

maṁ pariciyaṁ pi ujjhiya dullahalambhē jaṇammi saṅkanta!

jaggirasōhaggāṇaṁ aviṇayavāṇī vi varisaē harisaṁ

sō atthō jō hatthē, taṁ mittaṁ jaṁ nirantaraṁ vasaṇē

sō suhaō paramappuvvillanihāṇaṁ va naṭṭhasallaṁ va

diṭṭhaṁ siṇiddhanayaṇaṁ vayaṇaṁ viṇaōvarōhasōhillaṁ

ālavaṇaṁ savvaṅgassa bandhaṇaṁ tassa kiṁ bhaṇimō? // SubhāPajjSaṁ.28

gaṅgā uṇa tēṇa viṇā bhaṇa kattha khaṇaṁ pi vīsamaü? // SubhāPajjSaṁ.32

jattha'tthi gharaṁ ta ciya, annaṁ rannaṁ ti mannanti // SubhāPajjSaṁ.33

tāṇa kulavāliyāṇaṁ na lahaï saccaṁ sahāvamahimāṇaṁ
jāṇa paḍivakkhacaḍiyā dōsā vi maṇaṁ na dūminti // SubhāPajjSaṁ.35

tāṇa namō samaṇīṇaṁ ājammaṁ cariyavambhacērāṇaṁ
pimmanaḍakavaḍanāḍayaviḍaṁvaṇā jāṇa na khaṇaṁ pi // SubhāPajjSaṁ.36

tīē tijayaṁ pi kīraü niuñchaṇaṁ picchulucchujaṭṭhīē
chittā vi pīliyā vi hu jā na hu mahurattaṇaṁ muaï // SubhāPajjSaṁ.37

mahilā pasāyagahilā sayampabhā hōi, na uṇa sappurisō
sasiṇā viṇā vi rayaṇī havaï, na divasō viṇā raviṇā // SubhāPajjSaṁ.38

mannāmi māmi! ki vā majjha kaē tujjha dukkaraṁ annaṁ?
suhayassamajjhajaṁ kira appā vi samappiō tumaē // SubhāPajjSaṁ.39

jūyaṁ dhaṇāṇa, vasaṇaṁ guṇāṇa, paḍipatthaṇōvayārāṇa

hiyaēsu tēsu sahasa tti dukkhamakkhaliyamalliyaï // SubhāPajjSaṁ.43

aïkiviṇō cēya tumaṁ jaṁ ikkammi vi guṇammi ghippantē
