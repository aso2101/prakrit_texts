[dumapupphiyā]

dhammō maṅgalamukkiṭṭhaṁ ahiṁsā sañjamō tavō
dēvā vi taṁ namaṁsanti jassa dhammē sayā maṇō // Dasavē.1

jahā dumassa pupphēsu bhamarō āviyaï rasaṁ
na ya pupphaṁ kilāmēi sō ya pīṇēi appayaṁ // Dasavē.2

ēmēē samaṇā muttā jē lōē santi sāhuṇō
vihaṅgamā va pupphēsu dāṇabhattēsaṇērayā // Dasavē.3

vayaṁ ca vittiṁ labbhāmō na ya kōi uvahammaï
ahāgaḍēsu rīyantē pupphēsu bhamarā jahā // Dasavē.4

mahugārasamā buddhā jē bhavanti aṇissiyā
nāṇāpiṇḍarayā dantā tēṇa vuccanti sāhuṇō tti bēmi // Dasavē.5

[sāmaṇṇapuvvayaṁ]

kahaṁ nu kujjā sāmaṇṇaṁ jō kāmē na nivāraē
paē paē visīyantō saṅkappassa vasaṁ gaō? // Dasavē.6

vatthagandhamalaṅkāraṁ itthīō sayaṇāṇi ya
acchandā jē na bhuñjanti na sē cāi tti vuccaï // Dasavē.7

jē ya kantē piē bhōē laddhē vi piṭṭhi kuvvaï
sāhīṇē cayaï bhōē sē hu cāi tti vuccaï // Dasavē.8

samāi pēhāi parivvayantō siyā maṇō nissaraī bahiddhā
na sā mahaṁ nōvi ahaṁ pi tīsē iccēva tāō viṇaïjja rāgaṁ // Dasavē.9

āyāvayāhī caya sōgamallaṁ kāmē kamāhi kamiyaṁ khu dukkhaṁ
chindāhi dōsaṁ viṇaïjja rāgaṁ ēvaṁ suhī hōhisi samparāē // Dasavē.10

pakkhandē jaliyaṁ jōiṁ dhūmakēuṁ durāsayaṁ
necchanti vantayaṁ bhottuṁ kulē jāyā aṅgadhaṇē // Dasavē.11

dhiratthu tē'jasōkāmī jō taṁ jīviyakāraṇā
vantaṁ icchasi āvēuṁ sēyaṁ tē maraṇaṁ bhavē // Dasavē.12

ahaṁ ca bhōgarāyassa taṁ ca'si andhagavaṇhiṇō
mā kulē gandhaṇā hōmō sañjamaṁ nihuō cara // Dasavē.13

jaï taṁ kāhisi bhāvaṁ jā jā dicchasi nārīō
vāyāviddhu vva haḍō aṭṭhiappā bhavissasi // Dasavē.14

tīsē sō vayaṇaṁ soccā sañjayāi subhāsiyaṁ
aṅkusēṇa jahā nāgō dhammē sampaḍivāiō // Dasavē.15

ēvaṁ karanti sambuddhā paṇḍiyā paviyakkhaṇā
viṇiyaṭṭanti bhōgēsu jahā sē purisuttamō // Dasavē.16

tti bēmi.

[khuḍḍiyāyārakahā]

sañjamē suṭṭhiappāṇaṁ vippamukkāṇa tāiṇaṁ
tēsimēyamaṇāiṇṇaṁ nigganthāṇa mahēsiṇaṁ // Dasavē.17

uddēsiyaṁ kīyagaḍaṁ niyāgamabhihaḍāṇi ya
rāibhattē siṇāṇē ya gandhamallē ya vīyaṇē // Dasavē.18

saṁnihī gihimattē ya rāyapiṇḍē kimicchaē
saṁvāhaṇā dantapahōyaṇā ya sampucchaṇā dēhapalōyaṇā ya // Dasavē.19

aṭṭhāvaē ya nālīē chattassa ya dhāraṇaṭṭhāē
tēgicchaṁ pāhaṇā pāē samārambhaṁ ca jōiṇō // Dasavē.20

sejjāyarapiṇḍaṁ ca āsandī paliyaṅkaē
gihantaranisejjā ya gāyassuvvaṭṭaṇāṇi ya // Dasavē.21

gihiṇō vēyāvaḍiyaṁ jā ya ajīvavattiyā
tattānivvuḍabhōittaṁ āurassaraṇāṇi ya // Dasavē.22

mūlaē siṅgabērē ya ucchukhaṇḍē anivvuḍē
kandē mūlē ya saccittē phalē bīē ya āmaē // Dasavē.23

sōvaccalē sindhavē lōṇē rōmālōṇē ya āmaē
sāmuddē paṁsukhārē ya kālālōṇē ya āmaē // Dasavē.24

dhuvaṇetti vamaṇē ya vatthīkammavirēyaṇē
añjaṇē dantavaṇē ya gāyabbhaṅgavibhūsaṇē // Dasavē.25

savvamēyamaṇāinnaṁ nigganthāṇa mahēsiṇaṁ
sañjamaṁmi ya juttāṇaṁ lahubhūyavihāriṇaṁ // Dasavē.26

pañcāsavapariṇṇāyā tiguttā chasu sañjayā
pañcaniggahaṇā dhīrā nigganthā ujjudaṁsiṇō // Dasavē.27

āyāvayanti gimhēsu hēmantēsu avāuḍā
vāsāsu paḍisaṁlīṇā sañjayā susamāhiyā // Dasavē.28

parīsahariūdantā dhūyamōhā jiindiyā
savvadukkhappahīṇaṭṭhā pakkamanti mahēsiṇō // Dasavē.29

khavittā puvvakammāiṁ sañjamēṇa tavēṇa ya
siddhimaggamanupattā tāiṇē parinivvuḍā // Dasavē.30

tti bēmi ।।.

iha khalu bhō! pavvaïēṇaṁ uppannadukkhēṇaṁ sañjamē araïsamāvaṇṇicittēṇaṁ ōhāṇuppēhiṇā anōhāiēṇaṁ cēva hayarassigayaṅkusapōyapaḍāgābhūyāiṁ imāiṁ aṭṭhārasa ṭhāṇāiṁ sammaṁ sampaḍilēhiyavvāiṁ bhavanti taṁ jahā — haṁ bhō ! dussamāē duppajīvī, lahussagā ittariā gihīṇaṁ kāmabhōgā, bhujjō ya sāibahulā maṇussā, imē ya mē dukkhē na cirakālōvaṭṭhāī bhavissaī, ōmajaṇapurakkārē vantassa ya paḍiyāiyaṇaṁ, aharagaïvāsōvasampayā, dullahē khalu bhō ! gihīṇaṁ dhammē gihivāsamajjhē vasantāṇa, āyaṅkē sē vahāya hōi, saṅkappē sē vahāya hōi, sōvakkēsē gihavāsē niruvakkēsē pariāē, bandhē gihavāsē mukkhē pariyāē, sāvajjē gihavāsē aṇavajjē pariyāē, bahusāhāraṇā gihīṇaṁ kāmabhōgā, pattēyaṁ punnapāvaṁ, aniccē khalu bhō! maṇuyāṇa jīviē kusagga jalabindu cañcalē, bahuṁ ca khalu pāvaṁ kammaṁ pagaḍaṁ, pāvāṇaṁ ca khalu bhō! kaḍāṇaṁ kāmāṇaṁ puvviṁ ducinnāṇaṁ duppaḍikantāṇaṁ vēittā mukkhō natthi avēyaïttā tavasā vā jhōsaïttā, aṭṭhārasamaṁ payaṁ bhavaï. [bhavaï ya ittha silōgō]. 

jayā ya cayaī dhammaṁ, aṇajjō bhōgakāraṇā
sē tattha mucchiē bālē, āyaïṁ nāvabujjhaī // Dasavē.507

jayā ōhāviō hōi, indō vā paḍiō chamaṁ
savvadhammaparibbhaṭṭhō, sa pacchā paritappaï // Dasavē.508

jayā a vandimō hōi, pacchā hōi avandimō
dēvayā va cuyā ṭhāṇā, sa pacchā paritappaï // Dasavē.509

jayā a pūimō hōi, pacchā hōi apūimō
rāyā va rajjapabbhaṭṭhō, sa pacchā paritappaï // Dasavē.510

jayā ya māṇimō hōi, pacchā hōi amāṇimō
seṭṭhivva kabbaḍē chūḍhō, sa pacchā paritappaï // Dasavē.511

jayā a thēraō hōi, samaïkkantajovvaṇō
macchovva galaṁ gilittā, sa pacchā paritappaï // Dasavē.512

jayā a kukuḍumbassa, kutattīhiṁ vihammaï
hatthī va bandhanē baddhō, sa pacchā paritappaï // Dasavē.513

puttadāra parikiṇṇō, mōhasantāṇa santaō
paṅkōsannō jahā nāgō, sa pacchā paritappaï // Dasavē.514

ajja āhaṁ gaṇī huntō, bhāviappā bahussuō
jaï'haṁ ramantō pariyāē, sāmannē jinadēsiē // Dasavē.515

dēvalōgasamāṇō ya, pariyāō mahēsiṇaṁ
rayāṇaṁ arayāṇaṁ ca, mahānarayasārisō // Dasavē.516

amarōvamaṁ jāṇiya sukkhamuttamaṁ, rayāṇa pariyāē tahārayāṇaṁ
niraōvamaṁ jāṇiya dukkhamuttamaṁ, ramejja tamhā pariyāyapaṇḍiē // Dasavē.517

dhammāu bhaṭṭhaṁ siriō vavēyaṁ, jannaggi vijjhāyami'vappatēyaṁ
hīlanti ṇaṁ duvvihiaṁ kusīlā, dāḍhuddhiyaṁ ghōravisaṁ va nāgaṁ // Dasavē.518

ihēva dhammō ayasō akittī, dunnāmadhejjaṁ ca pihujjaṇaṁmi
cuyassa dhammāu ahammasēviṇō, sambhinnavittassa ya heṭṭhaō gaī // Dasavē.519

bhuñjittu bhōgāiṁ pasajjha cēyasā, tahāvihaṁ kaṭṭu asañjamaṁ bahuṁ
gaïṁ ca gacchē aṇahijjhiaṁ duhaṁ, bōhī a sē nō sulahā puṇō puṇō // Dasavē.520

imassa tā nēraïyassa jantuṇō duhōvaṇīyassa kilēsavattiṇō
paliōvamaṁ jhijjaï sāgarōvamaṁ, kisaṅga puṇa majjha ima manōduhaṁ // Dasavē.521

na mē ciraṁ dukkhamiṇaṁ bhavissaï, asāsayā bhōgapivāsa jantuṇō
na mē sarīrēṇa imēṇa'vissaï, avissaī jīviyapajjavēṇa mē // Dasavē.522

jassēvamappā u havijja nicchiō, caejja dēhaṁ na hu dhammasāsanaṁ
taṁ tārisaṁ nō payalenti indiyā, uvinti vāyā va sudaṁsaṇaṁ giriṁ // Dasavē.523

iccēva sampassia buddhimaṁ narō, āyaṁ uvāyaṁ vivihaṁ viyāṇiyā
kaēṇa vāyā adu mānasēṇaṁ, tiguttiguttō jiṇavayaṇamahiṭhṭhijjāsi // Dasavē.524

tti bēmi.

[vivittacariyā]

cūliyaṁ tu pavakkhāmi, suyaṁ kēvalibhāsiyaṁ
jaṁ suṇittu supuṇṇāṇaṁ, dhammē uppajjaē maī // Dasavē.525

aṇusōyapaṭṭhiē bahujaṇaṁmi, paḍisōyaladdhalakkhēṇaṁ
paḍisōyamēva appā, dāyavvō hōukāmēṇaṁ // Dasavē.526

anusōyasuhō lōō, paḍisōō āsavō suvihiyāṇaṁ
aṇusōō saṁsārō, paḍisōō tassa uttārō // Dasavē.527

tamhā āyāraparakkamēṇa, saṁvarasamāhibahulēṇaṁ
cariyā guṇā ya niyamā ya, hunti sāhūṇa daṭṭhavvā // Dasavē.528

aniēsavāsō samuyāṇacariyā, annāyaüñchaṁ païrikkayā ya
appōvahī kalahavivajjāṇā ya, vihāracariyā isiṇaṁ pasatthā // Dasavē.529

āinnaōmāṇavivajjaṇā ya, ōsannadīṭṭhāhaḍabhattapānē
saṁsaṭṭhakappēṇa carejja bhikkhū, tajjāyasaṁsaṭṭha jaī jaejjā // Dasavē.530

amajjamaṁsāsi amaccharīyā, abhikkhaṇaṁ nivvigaïṁ gayā a
abhikkhaṇaṁ kāussaggakārī, sajjhāyajōgē payaō havejjā // Dasavē.531

na paḍinnavijjā sayaṇāsaṇāiṁ, sejjaṁ nisejjaṁ taha bhattapānaṁ
gāmē kulē vā nagarē va dēsē, mamattabhāvaṁ na kahiṁ pi kujjā // Dasavē.532

gihiṇō vēyāvaḍiaṁ na kujjā, abhivāyaṇaṁ vandana pūyaṇaṁ vā
asaṅkiliṭṭhēhiṁ samaṁ vasejjā, munī carittassa jaō na hānī // Dasavē.533

na yā labhejjā niuṇaṁ sahāyaṁ, guṇāhiyaṁ vā guṇaō samaṁ vā
ekkō vi pāvāiṁ vivajjayantō viharijja kāmēsu asajjamāṇō // Dasavē.534

saṁvaccharaṁ vā'vi paraṁ pamāṇaṁ, bīyaṁ ca vāsaṁ na tahiṁ vasejjā
suttassa maggēṇa carijja bhikkhū, suttassa atthō jaha āṇavēi // Dasavē.535

jō puvvarattāvararattakālē, sampikkhaī appagamappaēṇaṁ
kiṁ mē kaḍaṁ kiṁ ca mē kiccasēsaṁ, kiṁ sakkaṇijjaṁ na samāyarāmi ? // Dasavē.536

kiṁ mē parō pāsaï kiṁ ca appā, kiṁ vā'haṁ khaliyaṁ na vivajjayāmi ?
iccēva sammaṁ anupāsamāṇō, anāgayaṁ nō paḍibandha kujjā // Dasavē.537

jatthēva pāsē kaï duppaüttaṁ, kāēṇa vāyā adu mānasēṇaṁ
tatthēva dhīrō paḍisāharejjā, āinnaō khippamiva kkhalīṇaṁ // Dasavē.538

jassērisā jōga jiindiyassa, dhiīmaō sappurisassa niccaṁ
tamāhu lōē paḍibuddhajīvī, sō jīvaī sañjamajīviēṇaṁ // Dasavē.539

appā khalu sayayaṁ rakkhiyavvō, savvindiēhiṁ susamāhiēhiṁ
arakkhiō jāipahaṁ uvēi, surakkhiō savvaduhāṇa muccaï // Dasavē.540

tti bēmi.
