namiūṇa jiṇa-var’indē tiyasēsara-vandiē dhuya-kilēsē
viusajaṇa-bōhaṇatthaṁ dhuttakkhāṇaṁ pavakkhāmi // Dhutt.1

laliya-samiddha-jaṇavayā atthi purī surapuriṁ visēsantī
ujjuya-jaṇāhivāsā ujjēṇī nāma nāmēṇaṁ // Dhutt.2

tīsē uttarapāsē nāṇāviha-kusuma-gumma-laya-gahaṇaṁ
mahuyari-gaṇōvagīyaṁ ujjāṇaṁ nandaṇa-saricchaṁ // Dhutt.3

tatth’ujjāṇa-gihammī nāṇāviha-rūva-kamma-ṇijjuttā
dhuttāṇa sayāṇēgā samāgayā kaha vi hiṇḍantā // Dhutt.4

māyā-ṇiyaḍi-pahāṇā akajja-ṇicc’ujjuyā niraṇukampā
thī-bāla-vuḍḍha-vīsattha-ghāiṇō vañcaṇ’ēkkaparā // Dhutt.5

dhūv’añjaṇa-jōgēhi ya ōsōyaṇi-thambhaṇīhiṁ vijjāhiṁ
sara-vaṇṇa-bhēya-kusalā tēsi pahū pañc’ imē kamasō // Dhutt.6

mūlasiri kaṇḍarīō ēlāsāḍhō sasō ya khaṇḍavaṇā
ēkk’ēkkassa ya tēsiṁ dhuttāṇaṁ pañca-pañcasayā // Dhutt.7

dhuttīṇaṁ pañcasayā khaṇḍavaṇāē ya navari parivārō
savvāṇa vi tāṇa pahū mūlasirī lōya-vikkhāō // Dhutt.8

vaṭṭaï ya tāṇa kālō jalaya-samucchanna-sūra-sasi-maggō
indadhaṇu-vijju-gajjiya-jalayara-muhalō varisayālō // Dhutt.9

sattāha-duddiṇammi ya samantaō bhariya-vappa-kūva-sarē
kaya-cikkhilla-cilipphila-dussañcārāsu racchāsu // Dhutt.10

ēyārisammi kālē tē dhuttā sīya-vaddalābhihayā
bhukkh’attā bēnti tahiṁ kō amhaṁ dējja bhattaṁ ti // Dhutt.11

aha bhaṇaï mūladēvō jaṁ jēṇa suyaṁ va samaṇubhūyaṁ vā
sō taṁ kahēu savvaṁ miliyāṇ’ ēyāṇa dhuttāṇaṁ // Dhutt.12

jō taṁ na pattiējjā asacca-vayaṇaṁ ti tēṇa savvēsiṁ
dhuttāṇa samuiyāṇaṁ dāyavvaṁ bhatta-pāṇaṁ ti // Dhutt.13

jō puṇa purāṇa-bhāraha-rāmāyaṇa-sui-samuttha-vayaṇēhiṁ
taṁ vayaṇaṁ samatthittā mahāyaṇaṁ pattiyāvējja // Dhutt.14

sō dhuttāṇa ahivaī mahāmaī atthi sattha-ṇimmāō
mā dēu kiñci kassa vi iya bhaṇiē mūladēvēṇa // Dhutt.15

tē savvē viya bhaṇiyā iya hōu susōhaṇaṁ taē bhaṇiyaṁ
kintu tumaṁ paḍhamaṁ ciya kahēhi jaṁ tē samaṇubhūyaṁ // Dhutt.16

aha bhaṇaï mūladēvō jaṁ aṇubhūyaṁ maē taruṇa-bhāvē
taṁ nisuṇēha avahiyā kahijjamāṇaṁ sujuttīē // Dhutt.17

taruṇattaṇammi ahayaṁ icchiya-suha-sampayaṁ ahilasantō
dhārā-dharaṇ’aṭṭhāē sāmi-gihaṁ patthiō suiraṁ // Dhutt.18

chatta-kamaṇḍalu-hatthō panthaṁ vāhēmi gahiya-pacchayaṇō
mattaṁ pavvaya-mēttaṁ pēcchāmi ya gayavaraṁ intaṁ // Dhutt.19

mēham iva gulagulintaṁ pabhinna-karaḍā-muhaṁ mahāmattaṁ
daṭṭhūṇa vaṇagay’indaṁ bhaēṇa vēvanta-gattō haṁ // Dhutt.20

attāṇō ya asaraṇō kattha nilukkāmi haṁ ti cintantō
tō sahasā ya aïgaō kamaṇḍaluṁ maraṇa-bhayabhīō // Dhutt.21

aha sō vi matta-hatthī ūsaviya-karō sarōsa-ratt’acchō
majjhāṇumagga-laggō kamaṇḍaluṁ aïgaō sigghaṁ // Dhutt.22

tō haṁ bhaya-sambhantō samantaō vidduyaṁ palōyantō
hatthiṁ kamaṇḍalummī vāmōhēūṇa chammāsaṁ // Dhutt.23

gīvāe niggaō haṁ hatthī vi mamāṇumaggaō nintō
laggō vāl’agg’antē kuṇḍiya-gīvāe chiddammi // Dhutt.24

aham avi ya navari puraō gaṅgaṁ pēcchāmi raṅgira-taraṅgaṁ
phēṇa-ṇiyar’aṭṭahāsaṁ vaṇagaya-danta-kkhaya-taḍ’aggaṁ // Dhutt.25

ummī-sahassa-paüraṁ jhasa-mayara-ggāha-kumma-pariyariyaṁ
juvaï-hiyaya vv’ agāhaṁ uyahi vva sudūra-parapāraṁ // Dhutt.26

paham annaṁ alahantō tō haṁ isu-vēya-vāhiṇiṁ sigghaṁ
bāhāhiṁ samuttiṇṇō gōpayam iva bhārahiṁ viulaṁ // Dhutt.27

tō sāmi-gihaṁ gantuṁ chuha-taṇhā-parisahēhiṁ sahamāṇō
chammāsā sīsēṇaṁ dharēmi dhārā-dhar’aṭṭhāē // Dhutt.28

dhārēūṇa ya dhāraṁ payaō ahivandiūṇa mahasēṇaṁ
sampattō ujjēṇiṁ tubbhēhiṁ samaṁ ca miliō haṁ // Dhutt.29

tō jaï saccaṁ ēyaṁ tō mē hēūhiṁ pattiyāvēha
aha mannaha aliyaṁ taṁ dhuttāṇaṁ dēha tō bhattaṁ // Dhutt.30

aha bhaṇaï kaṇḍarīō kō bhaṇihī tumam asacca-vayaṇaṁ ti
bhāraha-purāṇa-rāmāyaṇāṇi purisō viyāṇantō // Dhutt.31

paribhaṇaï mūladēvō sō hatthī kuṇḍiyāe kaha māō
kaha bhamiō chammāsaṁ kamaṇḍalē tammi vaṇahatthī // Dhutt.32

suhuma-cchiddēṇa kamaṇḍalāo kaha niggaō ahaṁ sō ya
niggantuṁ vaṇahatthī vāl’agg’antē kahaṁ laggō // Dhutt.33

kaha gaṅgā uttiṇṇā bāhāhiṁ maē sudūra-parapārā
kaha chammāsaṁ dhariyā bhukkhiya-tisiēṇ’ udaya-dhārā // Dhutt.34

aha bhaṇaï kaṇḍarīō jaṁ summaï bhārahē purāṇē ya
taṁ jaï savvaṁ saccaṁ tō saccaṁ tujjha vi ya vayaṇaṁ // Dhutt.35

hatthī kamaṇḍalummī ahaṁ pi māō kahaṁ ti jaṁ bhaṇasi
ēttha diyāi-pasiddhaṁ vayaṇaṁ suṇa paccaya-ṇimittaṁ // Dhutt.36

bambhāṇassa muhāō vippā khattiya-jaṇō ya bāhāsu
ūrūsu niggayā kira vaïsā suddā ya pāēsu // Dhutt.37

bambhāṇassa sarīrē jaï māō ēttiō jaṇa-samūhō
tō kaha kamaṇḍalummī na māsi taṁ vaṇagaya-samaggō // Dhutt.38

annaṁ ca bambha-viṇhū uḍḍhaṁ ca ahō ya bē vi dhāvantā
antaṁ jassa na pattā vāsa-sahassēṇa divvēṇa // Dhutt.39

liṅgaṁ mahappamāṇaṁ kaha māyaṁ tass’ umā-sarīrammi
ēvaṁ jaï kuṇḍiyāē hatthī māō tti kō dōsō // Dhutt.40

annaṁ ca imaṁ suvvaï risiṇā vāsēṇa bhārahē ’bhihiyaṁ
jaha vēṇu-pavva-majjhē kīyaga-bhāiya-sayaṁ jāyaṁ // Dhutt.41

kīō vaṁsō bhaṇṇaï tahiṁ jāyā kīyaga tti bhaṇṇanti
kaha tēsiṁ uppattī virāḍa-rāy’agga-mahisīē // Dhutt.42

tīē vi natthi puttō tō sā risi-āsamammi risim ēkkaṁ
ārāhiuṁ payattā tēṇa vi sō sāhiō caruō // Dhutt.43

bhaṇiyā tēṇa ya risiṇā ēyaṁ caruyaṁ tumaṁ kuḍaṅgammi
bhuñjāhi pahiṭṭha-maṇā hōhī tē tattha putta-sayaṁ // Dhutt.44

sā taṁ paribhuttuṁ jē vaṁsa-kuḍaṅgammi tō gayā sagihaṁ
kassaï kālassa tahiṁ gaggali-ṇāmō risivarō tti // Dhutt.45

vaṁsī-jāla-paviṭṭhō caraï tavaṁ navari tattha paüma-sarē
accharasā ṇhāyantī risiṇā diṭṭhā vigaya-vatthā // Dhutt.46

khuhiyassa sukka-binduṁ paḍiyaṁ nālīe kīyagō paḍhamō
nava-ṇāya-sahassa-balō sañjāō paḍhama-bindummi // Dhutt.47

jaha jaha nijjhāi muṇī accharasaṁ pavara-rūva-lāvannaṁ
taha taha khiraï ya bīyaṁ jāyaṁ tahiṁ kīyagāṇa sayaṁ // Dhutt.48

vaṁsī-ṇālīo risī nikkhiviuṁ niyayam ālayaṁ pattō
sō cciya vaṁsa-kuḍaṅgaṁ rāyā rakkhāvaē niccaṁ // Dhutt.49

sañjāya-savva-gattā gahiyā dēvīe tuṭṭha-maṇasāē
taṁ tīē putta-sayaṁ bhaṇṇaï nālī-samuppannaṁ // Dhutt.50

jaï vaṁsa-samuppannaṁ bhāya-sayaṁ kīyagāṇa māi tahiṁ
tō kaha kamaṇḍalummī na māsi taṁ vaṇagaya-samaggō // Dhutt.51

vāsa-sahassaṁ gaṅgā jaḍāsu vāmōhiyā jaï harēṇaṁ
chammāsaṁ kuṇḍiyāē vimōhiō vaṇagaō saccaṁ // Dhutt.52

laggō vāl’agg’antē hatthī ahayaṁ ca niggaō kaha nu
jaṁ bhaṇasi taṁ suṇējjasu purāṇa-sui-āgayaṁ vayaṇaṁ // Dhutt.53

viṇhū jagassa kattā sō kira ēg’aṇṇavammi lōgammi
āgāsa-bhūmi-māruya-tasa-thāvara-jalaṇa-rahiyammi // Dhutt.54

naṭṭha-ṇarāsura-tiriē tārā-sasi-sūra-virahiē lōē
tappaï tavaṁ aṇantō jala-sayaṇa-gaō acint’appā // Dhutt.55

tassa kira paṅkaya-tthō bambhō nāhīe paüma-gabbhāō
daṇḍa-kamaṇḍalu-hatthō viṇiggaō paṅkayaṁ laggaṁ // Dhutt.56

ēvaṁ kamaṇḍalu-gīvāe niggaō jaï tumaṁ gaya-samaṁ pi
hatthī vāl’agg’antē laggō tō ēttha kim ajuttaṁ // Dhutt.57

jaṁ pi ya bhaṇasi guṇāgara kuṇḍiya-gīvāe niggaō kim ahaṁ
bhāraha-pasiddham ēttha vi vayaṇaṁ suṇa paccaya-ṇimittaṁ // Dhutt.58

divvaṁ vāsa-sahassaṁ bambhāṇō tappaī tavaṁ raṇṇē
tō khuhiyā tiyasa-gaṇā bhaṇanti kaha hōjja sē vigghaṁ // Dhutt.59

tō bhaṇaï suravar’indō thī-lōlō pasuvaī-vivāhammi
aggīkamma-ṇiuttō daṭṭhūṇa umaṁ lhasiya-cēlaṁ // Dhutt.60

khuhiō bīya-ṇisaggaṁ karēi vihuṇaï niyaṁsaṇaṁ tāhē
paḍiyaṁ bīyaṁ kalasē dōṇāyariō samuppannō // Dhutt.61

ērisiya-itthiyāhiṁ ēsō khubbhējja rūva-kaliyāhiṁ
kō itthīhiṁ na khuhiō muttūṇa jiṇaṁ mahāvīraṁ // Dhutt.62

gōyama-vasiṭṭha-pārāsarō ya jamayaggi-kāsavō-’gatthī
ahayaṁ ca surāhivaī harō harī tihuyaṇa-variṭṭhā // Dhutt.63

annē vi ēvam āī tava-satta-ṇihī jayammi vikkhāyā
vammaha-bāṇābhihayā itthīhiṁ karāviyā āṇaṁ // Dhutt.64

tā gantūṇaṁ sigghaṁ tilōttamā uttamā sura-vahūṇaṁ
khōhēu tava-visiṭṭhaṁ iṭṭhēṇa piyāmahaṁ vihiṇā // Dhutt.65

tō sā saṅgaya-vēsā tiyasāhiva-vayaṇa-gahiya-sandēsā
gantuṁ vibhussa puraō divvaṁ naṭṭaṁ payāsēi // Dhutt.66

jahaṇ’uvvahaṇa-bhamirayā ucchāliya-hāra-manthar’ucchōhā
ghaṇa-masiṇ’aṁsiya-thaṇahara-m-ukkampiya-pēlava-dugullā // Dhutt.67

laliya-paṇay’aṅgalaṭṭhī sakaḍakkh’ubbhēya-bhuya-layā-juyalā
rēhaï paṇaccamāṇī pavaṇēriya-campaya-laya vva // Dhutt.68

daṭṭhūṇa naccamāṇiṁ bambhā ēg’indiō vva saṁvuttō
nijjhāi aṇimis’acchō sēs’indiya-mukka-vāvārō // Dhutt.69

nāūṇa tassa bhāvaṁ tō sā saṅgaya-maṇōhara-suvēsā
vivihāharaṇa-vihūsā dakkhiṇapāsē ṭhiyā vihuṇō // Dhutt.70

tō taṁ apēcchamāṇō bambhāṇō maṇasi maṇasijābhihaō
taṁ daṭṭhūṇa sataṇhō karēi bīyaṁ taō vayaṇaṁ // Dhutt.71

ēttō ’varēṇa taïyaṁ caütthayaṁ kuṇaï uttarēṇaṁ sō
uḍḍhaṁ uppaïyāē pañcama-vayaṇaṁ uvari-huttaṁ // Dhutt.72

tō tassa pamattassa ya tilōttamāgaya-maṇassa ruddēṇaṁ
taṁ vayaṇaṁ pañcamayaṁ nahēṇa ukkhuṇṭiyaṁ sahasā // Dhutt.73

tō bambhā parikuviō dāhiṇa-hattha-ppaēsiṇīē u
uvvaṭṭēi nilāḍē asarisa-rōs’uṭṭhiyaṁ sēyaṁ // Dhutt.74

tatth’ uppannō sēē balavantō sēyakuṇḍalī nāma
bambhēṇ’ āṇattēṇaṁ ukkhittō saṅkarō tēṇaṁ // Dhutt.75

tō sō bhaya-sambhantō nāsantō bayariyāsamē pattō
tattha harī niyamatthō bhikkhaṁ dēhi tti saṁlattō // Dhutt.76

tā tēṇa takkhaṇaṁ ciya mukka-sirā appaṇō nilāḍammi
bambhāṇa-sira-kavālaṁ ruddēṇa samuḍḍiyaṁ hēṭṭhē // Dhutt.77

vāsa-sahassēṇa vi taṁ na bharijjaï tīe ruhira-dhārāē
taṁ ruhiram aṅgulīē pasuvaïṇā ḍōhiyaṁ navariṁ // Dhutt.78

bambhāṇa-sira-kavālē kēsava-ruhir’aṅgulīe ruddassa
tō rattakuṇḍaliṇarō tiṇhaṁ sañjōgaō jāō // Dhutt.79

sō ruddēṇ’ āṇattō pajujjhiō sēyakuṇḍalīe samaṁ
jujjhantāṇaṁ tāṇaṁ vāsa-sahassaṁ aïkkantaṁ // Dhutt.80

tō gēṇhiūṇa dōnni vi savvēhi suravarēhiṁ miliēhiṁ
dinnō sakkassa narō sūrassa samappiō bīō // Dhutt.81

bhaṇiyā bhāraha-kālē bhāraha-jujjhassa kāraṇ’aṭṭhāē
bharah’avayāra-kālē uvaṇijjaha maṇuya-lōyammi // Dhutt.82

tō kālē sampattē sūrō kuntīe rūva-ummattō
kaya-sañjōō tīē kucchīe jaṇēi taṁ gabbhaṁ // Dhutt.83

sannaddha-baddha-kavaō kuntī-kaṇṇēṇa niggaō kaṇṇō
tō kiṁ tumaṁ na nīsarasi kuṇḍi-gīvāe mūlasirī // Dhutt.84

gaṅgā aṇōra-pārā kaham uttiṇṇa tti bhaṇasi mē jaṁ tu
ēttha vi paccaya-jaṇaṇaṁ suṇēhi rāmāyaṇē vittaṁ // Dhutt.85

sīyā-paütti-hēuṁ pavaṇa-suō rāhavēṇa āṇattō
laṅkā-puriṁ aïgaō bāhāhiṁ mahōyahiṁ tariuṁ // Dhutt.86

diṭṭhāē sīyāē piya-paḍivattiṁ suṇittu tuṭṭhāē
bhaṇiō kaha tē uyahī tiṇṇō haṇuyanta sō bhaṇaï // Dhutt.87

bhavati cātra ślōkaḥ:

tava prasādāt tava ca prasādād bhartuś ca tē dēvi tava prasādāt
sādhūnatē yēna pituḥ prasādāt tīrṇō mayā gōṣpadavat samudraḥ // Dhutt.87-1

jaï tēṇa samuttiṇṇō tiriēṇa mahōyahī duruttārō
tō kiṁ taē nar’uttama na hōjja gaṅgā samuttiṇṇā // Dhutt.88

jaṁ bhaṇasi kaha nu dhārā chammāsā dhāriyā sirēṇa maē
ēttha vi mē suṇa hēuṁ diyāi-sui-āgayaṁ vayaṇaṁ // Dhutt.89

lōga-hiy’aṭṭhāē kira gaṅgā abbhatthiyā suravarēhiṁ
avayarasu maṇuya-lōyaṁ saggāō bhaṇaï sā tāhē // Dhutt.90

kō maṁ dhariuṁ sakkō nivaḍantiṁ bhaṇaï pasuvaī ahayaṁ
dhārēmi taō paḍiyā dhariyā sīsēṇa pasuvaïṇā // Dhutt.91

divvaṁ vāsa-sahassaṁ jaï dhariyā jaṇhavī umā-vaïṇā
tō kaha na dharēsi tumaṁ chammāsaṁ sirēṇ’ udaya-dhārā // Dhutt.92

uttama-purisō si tumaṁ vinnāṇ’āgama-guṇēhiṁ sampuṇṇō
niggaya-jasō mahappā vikkhāō jīva-lōyammi // Dhutt.93

aïsaïō mūlasirī kaṇḍariyaṁ bhaṇaï bhaṇasu ēttāhē
jaṁ diṭṭhaṁ jaṁ ca suyaṁ aṇuhūyaṁ jaṁ ca tē ihaïṁ // Dhutt.1

aha bhaṇaï kaṇḍarīō aviṇaya-puṇṇō mi āsi bālattē
ammā-pii-duddantō rōsēṇa gharāo nikkhantō // Dhutt.2

parihiṇḍantō ya ahaṁ pattō dēs’anta-saṇṭhiyaṁ gāmaṁ
gō-mahisa-ajā-ēlaya-khara-karaha-samāulaṁ muiyaṁ // Dhutt.3

ārām’ujjāṇa-vaṇēhiṁ sōhiyaṁ kusuma-phala-samiddhēhiṁ
ayalāpuri-sāricchaṁ bahu-ghara-saya-saṅkulaṁ rammaṁ // Dhutt.4

tassa bahumajjha-dēsē pēcchaṁ vaḍapāyavaṁ maṇ’abhirāmaṁ
mēha-ṇiuramba-bhūyaṁ saüṇa-sahassāṇa āvāsaṁ // Dhutt.5

tassa ya hēṭṭhē jakkhō kamaladal’akkhō guṇēhiṁ parikiṇṇō
sannihiya-pāḍihērō dēi varaṁ sō vara-tthīṇaṁ // Dhutt.6

jakkhassa tassa jattā vaṭṭaï bahujaṇa-samāulā muiyā
tatth’ ēi jaṇō muiō dhūva-balī-puppha-hattha-gaō // Dhutt.7

ṇhāya-pasāhiya-jimiō savvālaṅkāra-bhūsiya-sarīrō
nāṇāviha-vattha-dharō candaṇa-parivaṇṇaga-vilittō // Dhutt.8

tō haṁ sakōuhallō uvāgaō taṁ mahāyaṇa-samūhaṁ
jakkhassa kamē namiuṁ ramamāṇē ghōḍahē pēcchaṁ // Dhutt.9

sannaddha-baddha-kavayā gahiy’āuha-paharaṇā ya aïbahulā
kalayala-ravaṁ karēntā paḍiyā cōrā navari tattha // Dhutt.10

tō sō sabāla-vuḍḍhō sa-itthiō jaṇavaō sapasuvaggō
aha ghōḍaēhiṁ sahiō vāluṅkaṁ aïgaō savvō // Dhutt.11

tattha vi tē ghōḍahaē ramanta-pēcchā jaṇē ya muiya-maṇē
cōrā vi paḍiṇiyattā naṭṭhō gāmō tti jampantā // Dhutt.12

navari tahiṁ pasuvaggō caramāṇō āgaō suvīsatthō
ēkkāe pasūe taō ōiliyaṁ cibbhaḍaṁ sahasā // Dhutt.13

sā ayagarēṇa ghatthā sō vi ya ḍhēṅkīe navari ōiliō
sā tatth’ ēva nilīṇā tuṅgē vaḍapāyavē viulē // Dhutt.14

tassa ahē khandhārō navari ṭhiō rāiṇō ya matta-gaō
ḍhēṅkī-pāē jamiō vaḍa-pārōhō tti kāūṇa // Dhutt.15

āuñciyammi pāē kaḍḍhijjaï gayavarō gulagulintō
tō ravaï mēṇṭha-vaggō kēṇāvi gaō samukkhittō // Dhutt.16

sōūṇa tāṇa saddaṁ sampattā saddavēhiṇō jōhā
isu-cāva-gahiya-hatthā kalayala-rāvaṁ karēmāṇā // Dhutt.17

chinnā ya tīe paṅkhā sē sīsaṁ hatthiēhiṁ dakkhēhiṁ
sā vilavantī paḍiyā pavvaya-sihara vva mahi-vaṭṭhē // Dhutt.18

vīī-paramparēṇaṁ ghōlintō uyaya-majjhammi // Dhutt.32

ēg’aṇṇavaṁ jagam iṇaṁ pañcamahābhūya-pabbhaṭṭhaṁ // Dhutt.33

ēvaṁ-vihē jagammī pēcchaï naggōhapāyavaṁ aha sō
mandaragiri vva tuṅgaṁ mahāsamuddaṁ va vitthiṇṇaṁ // Dhutt.34

sampuṇṇa-sarīr’uyaō miu-maddava-kuñciya-suēsō // Dhutt.35

hatthō pasāriō sē risiṇā ēh’ēhi vaccha bhaṇiō ya
khandhē mamaṁ vilaggasu mā marihisi uyaya-vāhīē // Dhutt.36

tēṇa ya ghēttuṁ hatthō ōiliō sō risī taō tassa
pēcchaï uyarammi jayaṁ sasēla-vaṇa-kāṇaṇaṁ savvaṁ // Dhutt.37

divvaṁ vāsa-sahassaṁ kucchīē sō risī paribhamantō
antaṁ na c’ēva pattō viṇiggaō risivarō tattō // Dhutt.38

jaï dārayassa uyarē sasurāsura-māṇusaṁ jayaṁ māyaṁ
tō cibbhaḍammi gāmō kaha nu na māējja kaṇḍariya // Dhutt.39

ḍhēṅkōyarē ayagarō tassa pasūē ya cibbhaḍaṁ uyarē
tattha vi ya jaṇa-samūhō kaha māō bhaṇasi suṇasu imaṁ // Dhutt.40

muṭṭhī-gējjha-sumajjhāe kēsavō dēvaīe kucchimmi
vuttō tassa ya uyarē sasēla-vaṇa-kāṇaṇā puhaī // Dhutt.41

aha bhaṇaï kaṇḍarīō cibbhaḍa-pasu-ayagar’āi-majjhammi

āvāha-vivāhā vi ya ussava-pasavā ya vaṭṭanti // Dhutt.44

tō luya-paṅkhō paḍiō mandara-siharō vva mahi-vaṭṭhē // Dhutt.63

sīyāē sō bhaṇiō sīlavaīṇaṁ saīṇa tēēṇaṁ
daṭṭhūṇa rāma-dūyaṁ hōhinti puṇō vi tē paṅkhā // Dhutt.64

kassaï kālassa taō haṇuyantō rāhavēṇa āṇattō

suiraṁ sāmatthēuṁ tihuyaṇa-ujjōya-kāraō jalaṇō
mahur’akkhara-vayaṇa-payampiēhiṁ bhaṇiō suragaṇēhiṁ // Dhutt.56

ēkkassa vi tā kīraï kajjaṁ abbhatthiēhiṁ jiya-lōē

vēsāṇarēṇa bhaṇiyaṁ kō dhāvaï abhimuhaṁ muhuttaṁ pi

vahiō uddhuya-liṅgō kō taṁ vivuhō samālavaï // Dhutt.62

balavantō vajjaharō tassa vi nissaṁsayaṁ bhāi // Dhutt.63

gaya-turaya-purisa-damaṇō kiṁ ca imō āgamō na suō // Dhutt.66

hatthī dammaï saṁvaccharēṇa māsēṇa dammaï turaṅgō
mahilā puṇa kira purisaṁ damēi ēkkēṇa divasēṇaṁ // Dhutt.67

jaṁ bhaṇaï umā-dēvī karēi taṁ pasuvaī akajjaṁ pi
kiṁ vā dēhāṇugayaṁ umaṁ vahantō na diṭṭhō tē // Dhutt.68

uṭṭha(?) ni-y-añciya-vayaṇaṁ piba rēyaṁ mā karēhi vikkhēvaṁ

kōmārabambhayārī n’icchaï maṇasā vi juvaï-sañjōyaṁ
savva-jaṇammi pagāsō dakkhiṇa-dēsē ṭhiō raṇṇē // Dhutt.83

caüsu ya disāsu jahiyaṁ dhāvaï lōgō bhavantara-dis’aṭṭhā

jaï mahasēṇ’aṅgāiṁ piha-gabbha-viṇiggayāiṁ laggāiṁ
tō tujjha na laggējjā tāiṁ kim ēg’uyara-vatthāiṁ // Dhutt.85

paramāṇu-pamāṇāō pagampiō laggasi tumaṁ pi // Dhutt.86

tō bhaṇaï ēlasāḍhō purisō kaha chinnaēṇa sīsēṇaṁ
bhukkh’attō bayarāiṁ khāēuṁ sakkaē kaha nu // Dhutt.87

bhaṇaï sasō rāhū kira sīsa-cchinnō jayammi sui-vāō
taha viya gayaṇē hiṇḍaï ābhaṁsaï canda-sūrē ya // Dhutt.88

aha bhaṇaï ēlasāḍhō kaha gammaï taṁ vigiṭṭham addhāṇaṁ
kaha vā vi jōyaṇa-sayaṁ kamēhiṁ akkamaï bhūmīē // Dhutt.89

paḍibhaṇaï sasō jannē balissa viṇhū diyāi-vēsēṇa

jaï savvā vi vasumaī tiṇṇi na puṇṇā kamē mahumahassa

lakkhaṇakumāra-paḍaṇē haṇuēṇaṁ dōṇa-pavvaō tuṅgō

mahaï-silā-saṅghāō sēlō jaï vāṇarēṇa ukkhittō
jōyaṇa-pamāṇa-mēttaṁ ukkhivasi silaṁ na sandēhō // Dhutt.95

lōē vi payaraï suī vaḍḍhantī mēiṇī mahumahēṇaṁ
kāuṁ varāha-rūvaṁ sa-sēla-vaṇa-kāṇaṇā dhariyā // Dhutt.96

jaï tēṇa samukkhittā na najjaī katthaī ṭhiēṇaṁ ti

ēlāsāḍhō jāhē sasēṇa aïsandhiō bhaṇaï tāhē

kāsava-risi-pattīō kaḍḍū viṇayā ahēsi tīyammi

païdivasaṁ kaḍḍūē viṇayā khiṁsijjaē ēvaṁ // Dhutt.46

garuḍēṇa jampiyaṁ bhukkhiō mi bhaṇiyaṁ ca kāsavēṇ’ ēvaṁ

dhagadhagadhaganta-huyavaha-pajjaliy’āvēḍhiaṁ samantēṇaṁ

sappi-mahō-dahi-salil’āiēṇa santappiē ’ṇalē dhaṇiyaṁ

kāsava-risi-vayaṇēṇaṁ gantuṁ garuḍēṇa dō vi sampayayā

rūviṇiyāṇa thambhiya-macchiya-saṅghāya-bhiṇibhiṇāyantaṁ
maḍaya-ciy’antōvari-ḍajjhamāṇa-silisilisilāyantaṁ // Dhutt.85

vijjā-sāhaṇa-rabhas’uṭṭhiyanta-ṇaccanta-maḍaya-vēyālaṁ
bhīm’aṭṭahāsa-huṅkāra-tāla-saddāla-gambhiraṁ // Dhutt.86

vasa-ruhira-kkaya-kaddama-samantaō bhīma-kimi-kul’āiṇṇaṁ
āmisa-kajja-samujjaya-ṇiṭṭhaya-bahuvihaga-jujjhantaṁ // Dhutt.87

ruhir’uggāla-bhalabhalaṁ sahassa-sūlā-vibhinna-mucchantaṁ
vitiriccha-bhamanta-siyāla-kōla-ṇivaḍanta-ghuṭṭantaṁ // Dhutt.88

aïduggandhiya-vāyanta-māruyaṁ sava-savanta-ṇīsaddaṁ
bībhacch’uvvēvaṇayaṁ bhaya-sañjaṇayaṁ surāṇaṁ pi // Dhutt.89

ēvaṁ vihē masāṇē pēcchaï aviṇaṭṭhayaṁ vigaya-jīvaṁ
acira-vimukkaṁ bālaṁ khaṇḍā taṁ gēṇhiuṁ ṇhavaï // Dhutt.90

abbhaṅgēūṇa tayaṁ jara-cīvara-susaṅgayaṁ karēūṇaṁ
ujjēṇīe paviṭṭhā sēṭṭhissa gharaṁ dhaṇa-samiddhaṁ // Dhutt.91

diṭṭhō ya ṇāe sēṭṭhī āsaṇa-viṭṭhō jaṇēṇa parikiṇṇō
bhaṇiō ya nāe bhāō suttiya-dhūyā mi duggaïyā // Dhutt.92

kaïvaya-divasa-pasūyā abandhavā asaraṇā vidēsa-tthā
tubbhē mahappabhāvā pii-hara-mullaṁ mamaṁ dēha // Dhutt.93

sēṭṭhī vāula-cittō puṇō puṇō tīe uccarantīē
rusiō bhaṇēi purisē sigghaṁ nīṇēha damiaṁ tti // Dhutt.94

niggacchasu tti bhaṇiyā aha jampaï mā chiv’ assa bālassa
annaṁ ṭhāṇaṁ bappi-kkayaṁ ti tō mē ma pēllēha // Dhutt.95

niggacchēuṁ n’icchaï tēhiṁ ya purisēhiṁ pēlliyā sahasā
dharaṇīyalē nivaḍiā bhaṇaï mahaṁ māriō puttō // Dhutt.96

hā majjha aṇāhāē nāhō hōhi tti cintayantīē
sō vi maṇōraha-tantū chinnō nicchitta-gattēhiṁ // Dhutt.97

bhō pēcchaha jaṇa samudaya imēṇa dhaṇa-gavviēṇa vaṇiēṇa
aṭṭhārasa-dōsa-vivajjiyāe mārāviō puttō // Dhutt.98

aha paharium āraddhā sīsē ya urē ya sā asāhāraṁ
bhaṇaï ya sēṭṭhī majjhaṁ bhaggaṁ bhikkhā-kavālaṁ tti // Dhutt.99

tō seṭṭhī ādannō savva-payattēṇa pariyaṇa-samaggō
aṇuṇēi vilavamāṇiṁ karēha mā suyaṇu bōlaṁ ti // Dhutt.100

dinnā ya kaṇṇiyā sē bhaṇiyā ghēttūṇa vaccasū puttaṁ
mā ruyasu mā ya kandasu tuha ēttiya jīvaṇaṁ dinnaṁ // Dhutt.101

ghēttūṇa kaṇṇiyaṁ mayaṁ kalēvaraṁ ca sā taō aïkkantā
sēṭṭhissa nirābāhaṁ jāyaṁ dāṇa-ppabhāvēṇaṁ // Dhutt.102

api ca. 

tyāgēna bhūtāni vaśībhavanti tyāgēna vairāṇy api yānti nāśam
parō ’pi bandhutvam upaiti dānāt tyāgō hi sarvavyasanāni hanti // Dhutt.103

sisu-maḍayaṁ chaḍḍēuṁ khaṇḍā viul’attha-lāha-parisuddhā
maṇi-kaṇaya-rayaṇa-mōttiya-camara-samiddhaṁ gayā haṭṭaṁ // Dhutt.104

kāūṇa ya viṇiōyaṁ tēsiṁ dhuttāṇa sīya-vihurāṇaṁ
bahu-khajja-pējja-kaliyaṁ susakkayaṁ bhōyaṇaṁ dēi // Dhutt.105

bhuttāyantēhiṁ taō savvēhiṁ vi haṭṭha-tuṭṭha-maṇasēhiṁ
bhaṇiyā ya khaṇḍavāṇā sujīviyaṁ jīviyaṁ tujjha // Dhutt.106

jaṁ tē buddhi-balāō dhutta-jaṇō nijjiṇittu sayarāhaṁ
santappiō khuh’attō viulēṇaṁ bhatta-pāṇēṇaṁ // Dhutt.107

sussikkhiyā vi purisā tāiṁ na jāṇanti jampiyavvāiṁ
jāiṁ asikkhiāō kattō vi lahanti mahilāō // Dhutt.108

paḍhiūṇa ya satthāiṁ purisā nāūṇa tēsim atthāiṁ
na samatthā paḍivayaṇē uppanna-maī jahā mahilā // Dhutt.109

api ca. 

adhītya śāstrāṇi vimr̥śya cārthān na tāni vaktuṁ puruṣāḥ samarthāḥ
yāni striyaḥ pratyabhidhānakālē vadanti līlāracitākṣarāṇi // Dhutt.110

cand’inda-vāu-sūrā aggī dhammō ya lōya-vikkhāyā
lōēṇa dūmiyā tē vammaha-raï-rāga-dōsēhiṁ // Dhutt.111

suvvaï ya āgamammī jaha kaṇhō savva-bīya-majjha-gaō
suhumēsu bāyarēsu ya tila-tusa-mēttēsu davvēsu // Dhutt.112

jaï savvagaō kaṇhō cintijjaï jattha tattha sō c’ēva
cintēntaō vi sō cciya tamhā sō kiṁ vicintēi // Dhutt.113

annaṁ pi aliya-vayaṇaṁ suvvaï lōyammi niggayaṁ iṇamō
jaha pavaṇa-gaṇāhivaī sēlasuyāvayava-uppannō // Dhutt.114

bambhāṇa-samuppattī tilōttamā-uvvasī ya dōṇassa
uppatti chammuhassa ya narakubbara āsi tāṇaṁ ca // Dhutt.115

kaṇhassa ya niggamaṇaṁ jaha kōvā sēyakuṇḍalī jāō
jaha sirakavāla-majjhē ruhirammi narō samuppannō // Dhutt.116

jaï jāyavassa māyā-uppattī halaharassa lōgammi
jaha jāyā sēlasuyā vikkhāyā jīva-lōyammi // Dhutt.117

jaï hōnti pavvayāṇaṁ puttā dhūyā kuḍumba-dhammō vā
tō taṁ imammi lōē jambūdīvē na māējjā // Dhutt.118

ēyāiṁ capphalāiṁ bhāraha-rāmāyaṇē nibaddhāiṁ
sañcālaṇam asahantā jaha jutti-kayaṁ suvaṇṇa vva // Dhutt.119

ēyaṁ lōiya-satthaṁ gaddaha-liṇḍa vva bāhirē maṭṭhaṁ
jāv’ antaṁ jōijjaï tusa-bhusa-busa-mīsiyaṁ savvaṁ // Dhutt.120

tō tē bhaṇāmi savvē kusamaya-kussui-pahēṇa muttūṇa
savvanna-dēsiyammi ya laggaha maggē payattēṇaṁ // Dhutt.121

ēyaṁ dhuttakkhāṇaṁ sōūṇaṁ lōiyassa param’atthaṁ
taha kuṇaha nicchiya-maïṁ jaha darisaṇa-suddhi hōi parā // Dhutt.122

cittaüḍa-dugga-siri-saṇṭhiēhiṁ sammatta-rāya-rattēhiṁ
sucariya-samūha-sahiyā kahiyā ēsā kahā suvarā // Dhutt.123

sammatta-suddhi-hēuṁ cariyaṁ haribhadda-sūriṇā raïyaṁ
nisuṇanta-kahantāṇaṁ bhava-virahaṁ kuṇaü bhavvāṇaṁ // Dhutt.124

sēyambara-vara-sūrī haribhaddō kuṇaü amha bhaddāiṁ
jassa sasi-saṅkha-dhavalē jiṇ’āgamē ērisā bhattī // Dhutt.125
