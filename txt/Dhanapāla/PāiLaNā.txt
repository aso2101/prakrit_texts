khattaṁ khannaṁ sāhū suyaṇō khēyāluō asahō // PāiLaNā.225

lahudāru kiliñcaṁ mantharā kusumbhī turaṅgiyā vaḍavā
saṅkhō kambū gēyaṁ gandhavvaṁ bandhaṇaṁ biṇṭaṁ // PāiLaNā.226

sīmā mērā vāsaṁ vuṭṭhī hālāhalō ya bambhaṇiyā
kārū sippī karahō kamēlaō rōhiō rojjhō // PāiLaNā.227

kēūram aṅgayaṁ viḍḍumaṁ pavālaṁ kaṇēṭṭhiyā guñjā
jhāḍaṁ kuḍaṅgaō kalusam āvilaṁ sitthayaṁ mayaṇaṁ // PāiLaNā.228

bukkā muṭṭhī saggaham uvarattaṁ loṭṭhayaṁ suvantaṁ ca
paratīraṁ pāraṁ aṭṭhayakkaliṁ jāṇa kaḍikhambhaṁ // PāiLaNā.229

lōhaṁ kālāyasam ummuyaṁ alāyaṁ visēsaō tilaō
kōṇō laüḍō jaüṇā kālindī gajjiaṁ thaṇiyaṁ // PāiLaNā.230

duvvā hariyālī koṭṭhaō kusūlō kaḍucchuō davvī
bhē tubbhē kullaḍayaṁ kōḍayam appullayaṁ niyayaṁ // PāiLaNā.231

ḍōlā piṅkhā māyaṇihayā jhalā pāusāgamō jhañjhā
duṭṭhō kalī payaṅgō salahō kalahāiyaṁ raḍiyaṁ // PāiLaNā.232

sāhariyaṁ sāhaṭṭhiyam aṁsō bhāō pihāṇiyā maṇḍī
tāraṁ dittaṁ vēsō nēvatthaṁ makkhiyaṁ tuppaṁ // PāiLaNā.233

jāṇa kaḍakkham avaṅgaṁ kalahaṁ parivāram ubbhayaṁ uddhaṁ
sāyaṁ paōsam uttappam uddhayaṁ dūsahaṁ tivvaṁ // PāiLaNā.234

dārāĩ duvārāiṁ nēlacchō paṇḍaō jaḍaṁ sisiraṁ
kōḍī aggaṁ paṅgū paṅgulaō taṇṇaō vacchō // PāiLaNā.235

chāyā chāhī pāhuḍam uvāyaṇaṁ sāriyā sālahiyā ya
bhandaṁ sivaṁ sayajjhō samōsiō chimpaō uñchō // PāiLaNā.236

sālō pāyārō majjiyā rasālāu sojjhaō rayaō
hariyaṁ saddalam aṅkō ucchaṅgō akkhayaṁ aṇahaṁ // PāiLaNā.237

maṇayaṁ īsi kuvindō ya tantuvāō durōyaraṁ jūyaṁ
pajjā ahiyārō aïram aṅgaṇaṁ majjiyaṁ ṇhāyaṁ // PāiLaNā.238

āsatthō vīsantō vajjhō vaddhō payaṭṭayaṁ caliyaṁ
karaṇī rūvaṁ caliyaṁ vaïkaliyaṁ parisarō pāsā // PāiLaNā.239

āraddhaṁ āḍhattaṁ nihayaṁ nikkhayam avēsiṇō phalihā
pharisō phaṁsō tumulaṁ muhularavō pāyasō khīrī // PāiLaNā.240

ummalaṇaṁ uccalaṇaṁ iharā annaha sahāsaō sabbhō
cittaṁ māṇasam akkhō bahēḍaō saṅgamō mēlō // PāiLaNā.241

samaō mayaṁ virāyaṁ vilīṇam uppāhalaṁ ca ukkaṇṭhā
sañcārī dūī sālirakkhiyā kalam agōvitti // PāiLaNā.242

ruggaṁ bhaggaṁ maṅgulam asundaraṁ vēviyaṁ ca tharahariyaṁ
papphōḍiyaṁ ca pakkhōḍiyaṁ visaṭṭaṁ vihaḍiyatthē // PāiLaNā.243

parihatthō dacchō kōvaṇō asahaṇō paīvaō dīvō
iya ēvaṁ rēsaṇiyā karōḍiyā candimā juṇhā // PāiLaNā.244

pāḍikkaṁ pattēyaṁ umbaraō dēhalī vihī divvaṁ
tantū guṇō dugañchā garihā khaliyaṁ paḍipphaliyaṁ // PāiLaNā.245

āsārō rayavuṭṭhī pamhalayaṁ rōmasaṁ jaō vēō
kīsa kiṇō halahalaō tarā viyaṭṭhaṁ visaṁvaïyaṁ // PāiLaNā.246

sambāhaṁ saṅkiṇṇaṁ valaggam ārūḍham ahiyam aïrittaṁ
āhūō vāhittō viyatthaṇaṁ niyaguṇasalāhā // PāiLaNā.247

jāṇa savāyaṁ ḍombaṁ soṇḍaṁ majjavam ayaṇḍam aṇavasaraṁ
saraḍaṁ ca kaṅkalāsaṁ chaṇaṁ mahaṁ loddhayaṁ vāhaṁ // PāiLaNā.248

lakkhaṁ vijāyam iṅgiyam āyārō aha imō imā ajjā
thāhō thagghō tōṇō tōṇīrō kaṇḍuō guliō // PāiLaNā.249

kaṇṭhō galaō avaḍū kiyāḍiyā jhampaṇīu pamhāiṁ
gupphā khuluhā jaṅghā ṭaṅkā gaṇḍā kavōlā ya // PāiLaNā.250

rasaṇā jīhā savaṇā kannā vacchaṁ uraṁ bhuyā bāhū
bhumayā bhamuhā kakkhā bhuyamūlaṁ jaṇhuyā jāṇū // PāiLaNā.251

kannā kumarī dhūyā duhiyā bahiṇī sasā piyā jaṇaō
māyā jaṇaṇī suṇhā puttavahū dēyarō diyarō // PāiLaNā.252

māucchā māusiyā attā sāsū sahōyarō bhāyā
mallāṇī māmī pupphiyā piucchā paī bhattā // PāiLaNā.253

rambhā kayalī sindī khajjūrī sattalā ya nōmālī
gummō jālī bōrī kakkandhū kēsarō baülō // PāiLaNā.254

palahī vavaṇaṁ tūlō rūvō dabbhō kusō javō gajjō
thambā saḍhā kalambō nīvō gōlhāphalaṁ bimbaṁ // PāiLaNā.255

aïmuttō māhaviyā lavaō gundō piyaṅguṇō kaṅgū
kalamō sālī ya bisaṁ muṇālam ullūḍham aṅkuriyaṁ // PāiLaNā.256

naggōhaṁ vaḍarukkhaṁ sigguṁ sōhañjaṇaṁ talaṁ tālaṁ
cāraṁ piyālam ajuyalavannaṁ sattacchayaṁ jāṇa // PāiLaNā.257

bīyayam asaṇaṁ pippalam āsatthaṁ tiṇḍuyaṁ ca ṭimbaruyaṁ
rāyañchuyaṁ ca vēḍisamalambusaṁ bōḍaghēraṁ ca // PāiLaNā.258

khaliṇaṁ kaviyaṁ maïlaṁ malīmasaṁ sīharā jalatusārā
thāṇū khaṇṇuyam alayā kuralā dantacchaō uṭṭhō // PāiLaNā.259

sīlaṁ payaī tattaṁ paramatthaṁ gōuraṁ paōlī ya
rayaṇī hatthō bhīō hitthō sāhīṇam appavasaṁ // PāiLaNā.260

dīṇō varaō viḍḍhiram āḍōvō raggayaṁ ca navaraṅgaṁ
laṇhaṁ masiṇaṁ ṭhāṇaṁ ōvāsō gōyarō visaō // PāiLaNā.261

ārōviyaṁ vilaïyaṁ dhōyaṁ vicchōliyaṁ dasā vatthā
saṅkū kīlō ruṇṭiyam aliviruyaṁ sīriō bhinnō // PāiLaNā.262

asamañjasam anibaddhaṁ paḍisiddhō vāriō carō paṇihī
nihasō kasō samantā savvattō cāraō kārā // PāiLaNā.263

vihulaṁ visaṇṭhulaṁ jāṇa sāmariṁ simbaliṁ thaliṁ bhūmiṁ
gharavāḍayaṁ parōhaḍam āyaïmajjantakālaṁ ca // PāiLaNā.264

vāliyayaṁ pariyattiyam ūḍhiyayaṁ pāuyaṁ miyaṁ tucchaṁ
huṇḍī ghaḍā paüṭṭhō kalāiyā gaṇḍaō khaggō // PāiLaNā.265

saṇḍaṁ vaṇaṁ muyaṅgō muraō ya vivajjaō vivajjāsō
pamahā gaṇā kalāvō tiuḍō vatthaṁ dugullaṁ ca // PāiLaNā.266

mahuraṁ sāuṁ vārī karidharaṇaṭṭhāṇam aggalā phalihō
visayaṁ phuḍaṁ tarantaṁ parippavantaṁ hiyaṁ nīyaṁ // PāiLaNā.267

lakkhāruṇiyaṁ pallaviyam āviyaṁ pōiyaṁ ghaḍhaṁ thūhaṁ
rūḍhaṁ paüṇaṁ jāmō paharō bahulō kasaṇapakkhō // PāiLaNā.268

paḍibhēō paccāraṇam āsandī pīḍhiyā raṇī saddō
sayaḍō gantī taṁsaṁ tiriccham asidhēṇuyā churiyā // PāiLaNā.269

chōhō vikkhēvō hiyayam āsaō kandarō ya kappāḍō
kūṇiyam addhanimīliyam acchāyantaṁ nisāyantaṁ // PāiLaNā.270

saṁmaddō saṅghaṭṭō rāsō hallīsaō khamaṁ uciyaṁ
gujjhaṁ rahassamāsā maṇōrahō kōsayaṁ casayaṁ // PāiLaNā.271

raṅgō picchābhūmī darī guhā cūḍaō valayabāhū
khañjō khōḍō ya naḍō kusīlavō nāliyā ghaḍiyā // PāiLaNā.272

pāsāō hammiyamaṇuyaghaṇṭiyā kiṅkiṇī kaṇī phuraṇaṁ
pōō vahaṇaṁ sabarā ya kirāyā mālaī jāī // PāiLaNā.273

jetti payapūraṇē vimhayaṁmi hī vilaviyaṁmi avi hāhā
antō majjhē puraō ya aggaō maggaō pacchā // PāiLaNā.274

khēyāisu avvō handi utti āmantaṇaṁmi dēsaddō
ira tacchīlē illō ittō ālō ya maüyatthē // PāiLaNā.275

vikkamakālassa gaē aüṇattīsuttarē sahassaṁmi
mālavanarindadhāḍīe lūḍiē mannakhēḍaṁmi // PāiLaNā.276

dhārānayarīe pariṭṭhiēṇa maggē ṭhiyāe aṇavajjē
kajjē kaṇiṭṭhabahiṇīe sundarīnāmadhijjāē // PāiLaNā.277

kaïṇō andha jaṇa kivā kusala tti payāṇam antimā vannā
nāmaṁmi jassa kamasō tēṇēsā viraïyā dēsī // PāiLaNā.278

kavvēsu jē rasaḍḍhā saddā bahusō kaīhi bajjhanti
tē ittha maē raïyā ramantu hiyaē sahiyayāṇaṁ // PāiLaNā.279
